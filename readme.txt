Pagina del desafio final del bootcamp Javascript Fullstack Trainee.

Activar el server.js para ver la pagina donde deseen.

Dejo una foto de referencia en la carpeta principal llamada "homepage example.jpg" porseacaso.


Esta página se genera de forma dinámica desde el servidor.

Supuestamente es un listado de participantes a una competencia de skate.

Permite guardar permanentemente usuarios/participantes con sus respectivos datos e imagenes,
que posteriormente pueden ingresar a la pagina, donde al validarlos activa una sección especial para administradores
donde pueden cambiar el estado de aprobado/espera de los participantes.
